第一次用Hexo+GitHub搭建博客

发表于 2014-11-22 更新于 2017-09-17 分类于 杂谈
本文字数： 588 阅读时长 ≈ 1 分钟

写博客是很有好处的

1.记录生活中的每分每秒
人生中的各种经历，各种想法，我们都可以把它写到博客上，这样，珍贵的片段就被记录起来，使得这些记忆、想法不会随着时间的流逝而冲谈，最后忘却所有。

2.锻炼表达能力
写博客文章就像是学生时代写作文一样，只不过以前只是为了应试而写，现在是为了自己的兴趣爱好而写。
不同类型的文章，表达方式也不一样，自己要更清楚地表达不同的内容，表达方式肯定要转变。文章写多了，就会促使表达能力的提高。

3.拓展思维
写博客是一种互动交流，看博客的人或许会留下评论，这样别人也会分享想法给自己，这样自己的思维就会更开阔。

在GitHub上搭建属于自己的博客

本来我是打算再csdn上写博客的，偶然间看到一篇关于GitHub的文章，才得知原来也可以在那上面写博客（看来我是火星了）。
GitHub上写博客，是很自由的。自己可以用别人的博客主题，自己有技术的话，同样是可以自制属于自己的博客主题，无论是博客中的样式还是交互功能，自己都可以根据自己的喜好来设计。
在这里，我就用了Hexo + GitHub搭建的博客，使用的是yilia这一个主题，或许某一天等我自己技术提高了，我也会自定义属于自己的博客主题。
无论你是否懂编程技术，其实都可以在GitHub上搭建属于自己的博客，在GitHub上搭建博客并不难。

就先这样吧~
