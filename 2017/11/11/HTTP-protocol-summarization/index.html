URI和URL

概念

URI

**统一资源标识符（Uniform Resource Identifier/URI）是一个可以唯一地标识某个资源的字符串。**
更通俗地理解就是：
URI是一个可以唯一地标识出某个事物（资源）的字符串，比如：身份证号可以唯一地标识出某个人，手机号码可以唯一地标识出某个SIM卡。
在这，我们就可以将身份证号，手机号码看作是生活中的URI，因为它们都是唯一地标识出某个事物。

URL

统一资源定位符（Uniform Resource Locator/URL）**俗称网址（web address），它是一个可以唯一地指定某个网络资源在网络上的位置以及检索机制的引用。**
